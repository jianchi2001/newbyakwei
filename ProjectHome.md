# mvc #
java mvc web框架，目前只支持jsp
上手速度快，学习成本低。
框架基础环境：Servlet Spring3 jee5以上版本。

本框架的目的是为了简单的创建并使用web开发，让开人人员脱离struts2繁琐的配置以及冗余的annotation代码。

你只需要进行简单的配置，即可使用该框架进行web开发。
